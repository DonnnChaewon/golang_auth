# How to run
Enter ```cmd /k code .``` in the folder directory.<br>
Open Laragon application -> Start All -> Database -> add new database named golang_auth.<br>
Open VSCode, open new terminal, and type ```go run .```<br>
Don't forget to CTRL + C in the terminal and ```go run .``` ever time after you changed the code.<br>
